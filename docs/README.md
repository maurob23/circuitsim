# CircuitSim Technical Docs For AI Collaboration

Updated: 2026-05-25

This folder contains an updated technical briefing for collaborators and AI
assistants working on CircuitSim.

Recommended reading order:

1. `architecture_overview.md`
2. `backend_structure.md`
3. `frontend_structure.md`
4. `api_spec.md`
5. `simulation_engine.md`

Use these files as the first context package when asking ChatGPT or another AI
model for architectural advice. The older `doc/` folder remains useful for user
manuals, roadmap notes, and help content, but some older technical statements may
refer to pre-registry frontend architecture.
